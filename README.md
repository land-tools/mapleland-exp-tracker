# 🍁 메이플랜드 경험치 측정기

바닐라 JavaScript로 구현된 메이플랜드 게임 화면 분석 도구입니다.
화면 캡처와 OCR을 통해 실시간으로 경험치와 골드 획득량을 추적합니다.

## ✨ 주요 기능

- **화면 캡처**: 게임 창을 선택하여 실시간 모니터링
- **영역 선택**: 마우스 드래그로 EXP/메소 영역 지정
- **OCR 인식**: Tesseract.js를 사용한 텍스트 추출
- **실시간 분석**: 5초 주기로 변화량 및 시간당 획득량 계산
- **PIP 오버레이**: Picture-in-Picture 창으로 게임 위에 결과 표시
- **레벨업 감지**: 자동으로 레벨업 감지 및 기준값 리셋
- **설정 저장**: LocalStorage로 영역 좌표 자동 저장

## 🚀 사용 방법

### 1. 실행
```bash
# 로컬 서버로 실행 (Live Server 등)
# 또는 index.html을 브라우저에서 직접 열기
```

> ⚠️ `file://` 프로토콜에서는 일부 기능이 제한될 수 있습니다. 로컬 서버 사용을 권장합니다.

### 2. 화면 선택
1. **[화면 선택]** 버튼 클릭
2. 메이플랜드 게임 창 선택
3. 프리뷰에서 게임 화면 확인

### 3. 영역 설정
1. **[EXP 영역]** 버튼 클릭 후 EXP 표시 부분 드래그
   - 예: `EXP 55289816 [19.79%]` 영역
2. **[메소 영역]** 버튼 클릭 후 메소 표시 부분 드래그
   - 예: `78,972,001 메소` 영역 (인벤토리 열린 상태)

### 4. 분석 시작
1. **[분석 시작]** 버튼 클릭
2. 5초마다 자동으로 OCR 분석 수행
3. **[PIP 열기]**로 게임 위에 결과 오버레이

## 🛠️ 기술 스택

- **HTML5 / CSS3 / Vanilla JavaScript**
- **Screen Capture API**: `getDisplayMedia()`
- **Canvas API**: 이미지 처리 및 크롭
- **Tesseract.js**: OCR 텍스트 인식
- **Picture-in-Picture API**: 오버레이 표시
- **LocalStorage**: 설정 저장

## ⚡ 레벨업 감지

퍼센트가 50% 이상 급감하면 레벨업으로 감지하고 자동으로 기준값을 리셋합니다.

예: 99.5% → 0.3% (레벨업!)

## 🔧 OCR 팁

더 정확한 인식을 위해:
- 게임 창이 선명하게 보이도록 해상도 설정
- EXP/골드 영역만 정확히 선택
- 다른 UI 요소가 겹치지 않도록 주의
